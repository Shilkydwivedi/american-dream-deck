# American Dream — Interactive Sales Deck

A cinematic, browser-based interactive sales deck for American Dream Mall.

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- No frameworks or dependencies
- YouTube embed for video

## Setup
Open index.html in any browser. No build step required.

## Design Decisions
- Luxury editorial aesthetic inspired by Hermès and Apple
- Non-linear scroll navigation
- Gold/obsidian color palette for premium feel
- Custom cursor, scroll-triggered animations, video modal

## AI Tools Used
- Claude (Anthropic) — built the entire deck
- Used for layout, copywriting, animation logic, and UX structure
